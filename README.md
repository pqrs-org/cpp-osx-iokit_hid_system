# cpp-osx-iokit_hid_system

IOKit type definitions.

## Requirements

cpp-osx-iokit_hid_system depends the following classes.

- [pqrs::osx::iokit_types](https://github.com/pqrs-org/cpp-osx-iokit_types)

## Install

### Using package manager

You can install `include/pqrs` by using [cget](https://github.com/pfultz2/cget).

```shell
cget install pqrs-org/cget-recipes
cget install pqrs-org/cpp-osx-iokit_hid_system
```

### Manual install

Copy `include/pqrs` directory into your include directory.
